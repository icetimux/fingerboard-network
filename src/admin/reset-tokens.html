Reset Tokens

When a user requests a password reset, a one-use token is emailed automatically. All recent requests are listed here for oversight. Tokens expire after 1 hour.

info

Password reset flow:

  1. User enters their username + email on the site.
  2. If they match, an email is sent automatically via Resend with the reset token.
  3. User enters their username + token + new password on the site.
  4. Token is consumed immediately and cannot be reused. Unused tokens expire after 1 hour.
